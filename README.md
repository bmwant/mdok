Tired of ...

[VIDEO HERE]

### Installation

```bash
$ npm i ok-run
```

### Features

* Hints/examples/default on parameters
* Validation for parameters entered


### Usage

* Invoke for the target command

```bash
$ ok tar
$ ok run tar  # equivalent
```

* List available receipes

```bash
$ ok list
```
