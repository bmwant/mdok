Usage: ok <recipe> [options]

General options:
  --help         # Print this info and generator's options and usage
  -f, --force    # Overwrite files that already exist
  -V, --version  # Print version
  --no-color     # Disable colors
  --confirm      # Ask confirmation before executing command

Run recipe:

  $ ok <recipe-name>
  $ ok curl

Completion:

  To enable shell completion for the ok command, try running

  $ ok completion

Full Documentation: https://github.com/bmwant/ok
